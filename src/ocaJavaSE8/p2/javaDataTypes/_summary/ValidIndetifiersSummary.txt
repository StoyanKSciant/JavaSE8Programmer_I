
* A valid identifier starts with letter (a-z, upper or lower case), a currency
sign, or an underscore. There is no limits to its length.

* A valid identifier can contain digits but not in the starting place.

* A valid identifier can use the underscore and currency sign at any
position of the identifier.

* A valid identifier can't have the same spelling as a Java keyword, suck as
switch.

* A valid identifier can't use any special characters, including:
		!, @, #, %, ^, &, *, (, ), ', :, ;, {, /, \, }.
